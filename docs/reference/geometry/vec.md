## Class `vec`

### Overview

Class `vec` implements an object-oriented interface to two-dimensional vectors, offering operators for all its elementary operations.

### Example usage

```c++
vec my_vec(42.42, 24.24);
my_vec.x += 22;
std :: cout << my_vec << std :: endl; // Prints (64.42, 24.24);

// Etc
```
